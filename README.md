## Setup

### Install pip and virtualenv

- `cd gistipedia/server`
- `virtualenv2 .`
- `bin/pip install -r requirements.txt`

### Running

- `cd gistipedia/server`
- `bin/python2 app.py`
- `localhost:5000`
